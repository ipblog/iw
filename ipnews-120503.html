谷歌甲骨文诉讼案值得关注的10个关键问题

投递人 itwriter 发布于 2012-05-10 10:44

据 eWeek 网站发表的一篇分析文章称，谷歌和甲骨文之间的诉讼还远远没有结束。陪审团在 5 月 7 日做出的部分裁决使这个案子更加复杂。观察人士需要跟踪许多要点以观察这个案子在这个裁决之后的发展。谷歌和甲骨文之间一直采取各种手段对 Java 和 Android 展开顽强的法律斗争。这个斗争还远远没有结束。 然而，一个陪审团在 5 月 7 日对这个案子做出了部分裁决，给予甲骨文一个小胜，尽管双方各现在都声称取得了胜利。

这个陪审团称，谷歌侵犯了甲骨文 Java 编程工具的版权，并且 Android 操作系统中只有 9 行代码侵犯了版权。除此之外，不允许甲骨文要求获得它一直要求的 10 亿美元赔偿。

这个裁决让人非常惊奇并且在移动领域产生了不确定性。从苹果到三星到谷歌和摩托罗拉，数不清的公司都争辩称一个竞争对手侵犯了他们拥有的专利或版权。到目前为止，每一个案子都没有能够达成一个最终的确定性的协议。无论是否喜欢，这些案子都将持续许多年。根据谷歌甲骨文的案子的初步裁决判断，这些案子不会很快地轻松地结束。下面是人们应该了解的有关谷歌甲骨文诉讼案的 10 个关键问题。

1.  这个案子完全是围绕 Java 的

虽然甲骨文起诉谷歌的案子目前是围绕 Android 发展的，但是，这个诉讼的真正核心问题是 Java。甲骨文争辩称，谷歌在 Android 操作系统中侵犯了 Java 版权（不是专利），而谷歌称它是无辜的。

2.  甲骨文要求大量现金赔偿

甲骨文要求法院判决谷歌因 Android 侵犯其知识产权赔偿 10 亿美元。然而，这个数字一直被认为是太多，即使甲骨文最终胜诉，甲骨文获得的赔偿也不会接近这么高的数字。

3.  谷歌一直愿意让步

虽然谷歌声称它是无辜的，但是，据说谷歌一直愿意让步，与甲骨文达成协议以阻止长期的、昂贵的和持续很久的法律诉讼。然而，甲骨文表示，这个协议条款是不能接受的，从而使两家公司进入了当前这个案子。

4.  谷歌失败

现在背景已经介绍完毕，我们来看看目前的状况。谷歌在这个案子的第一次裁决中失败了。陪审团发现谷歌确实在 9 行代码中侵犯了甲骨文的 Java 版权。陪审团还发现谷歌侵犯了 37 个 Java 应用程序编程接口的版权。因此，谷歌在这场战争的第一次战斗中失败了。

5.  谷歌取胜

不过，在甲骨文打开香槟酒庆祝之前，重要的指出谷歌在第一次裁决中也获胜了。这个裁决不允许甲骨文要求获得 10 亿美元赔偿。此外，陪审团还没有做出结论， 谷歌应用 Java 应用程序编程接口是不是“公正地使用”开源软件技术。因此，虽然甲骨文可能在原则上获胜，但是，谷歌可能在实践上获胜。

6.  Android 当前的功能面临危险

不过，这场斗争还远远没有结束。这意味着 Android 仍是不安全的。如果最终确定谷歌没有“公正地使用”甲骨文的 Java 应用程序编程接口，并且随后决定 Android 中将如何使用 Java，谷歌可能被迫修改 Android 以便继续提供这个软件。

7.  这个案子意义重大

一旦谷歌被迫修改 Android 以便遵守法院的命令，这个意义将非常重大。例如，Android 市场中的大多数应用程序都将无法使用，因为这些应用程序都依赖 Java。此外，谷歌将向市场上的每一个基于 Android 的设备发送更新软件。这是一个不小的工作量。

8.  甲骨文仍能得到一些现金

虽然甲骨文不能得到 10 亿美元损失赔偿，但是，甲骨文仍有机会从谷歌那里得到一些现金。事实上，根据这个裁决之后的赔偿律例，甲骨文可能因谷歌侵权获得 15 万美元赔偿。这笔赔偿金也许不够这个非常昂贵的法律诉讼的一天的成本。

9.  授权仍有可能性

虽然谷歌痛恨这种想法，但是，这个案子最终有可能以甲骨文和谷歌之间达成许可证授权交易而结束。总之，如果这个案子无限期拖延下去，对于谷歌来说简单地购买甲骨文的 Java 许可证也许会更便宜。此外，如果谷歌败诉，谷歌购买 Java 许可证要比修改 Android 更好一些。许可证授权仍有非常大的可能性。　

10.  这个案子还远远没有结束

最后，每一个人都应该认识到，谷歌与甲骨文之间的这个案子还远远没有结束。现在，双方好像都愿意与对方和好。诸如进行许可证交易谈判等简单的决定就可能结束这个案子。但是，在任何一方认识到他们在法院的裁决中不会得到自己要求的东西之前。这个简单的决定还不会出现。有一个非常真实的可能性，就是甲骨文与谷歌之间的案子可能会拖许多年。不相信吗？想想谷歌与美国媒体巨头维亚康姆（Viacom）之间的激烈的斗争。
