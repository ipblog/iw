安装指南

 

IPMis专利管理系统分为个人版和企业版。个人版采用的是Access本地数据库, 适合个人用户管理专利案件使用。企业版采用SqlServer数据库,适合企事业或小型事务所多人共同管理专利案件使用。IPMis专利管理系统个人网络版采用网络验证方式，个人单机版采用本地验证方式，企业版采用本地验证方式。

IPMis专利管理系统尽可能保持系统小巧,避免系统的臃肿。目前包含数据库等全部文件的安装包共约6.5M大小。在安装过程中所有文件均放在安装目录下，不往系统其它任何目录下安装任何文件。

安装过程如下：

1. 用户首先登陆http://www.ipwom.com/, 点击页面右上角注册链接，进入注册页面。

1

2. 在注册页面创建一个登录帐号，特别注意其中的 用户 ID 、邮件地址和 密码 三项。当注册完成并激活后 可用用户 ID/ 密码或 邮件地址/密码作为用户名/密码来登录IPMis专利管理系统。

2

3. 点击链接http://www.ipwom.com/ipmis-p-trial.aspx进入到IPMis专利管理系统试用下载页面, 点击页面最下的下载地址下载最新的IPMis专利管理系统。

3

4. 将下载的IPMis-Patent.zip压缩包解压, 得到IPMis-Patent目录。由于360软件经常误报，IPMis专利管理系统可能与360不兼容，如果出现问题建议把360临时关闭。

 

4

 5.右键IPMis目录内的IPMis.exe主程序, 选择创建快捷方式, 创建IPMis主程序的快捷方式, 我们可以改名为IPMis专利管理系统。

6. 将IPMis专利管理系统的快捷方式拷贝到用户桌面上。

5

6. 双击快捷方式运行IPMis专利管理系统，出现登录界面, 用 在第2步注册的用户ID/密码或邮件地址/密码作为用户名/密码(注意大小写)来登录IPMis专利管理系统。

如果用户名/密码没有错误，便进入IPMis专利管理系统的主界面。(注意：如果帐号不激活的话很有可能将来是登不进去的)

IPMis专利管理系统个人版安装成功！

如果在安装IPMis专利管理系统的过程中出现任何问题，欢迎到IPMis专利管理系统的论坛中发表。
