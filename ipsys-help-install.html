安装指南

Rate this Content 0
     
0 投票

 

IPSys商标管理系统是一款适合知识产权代理机构使用的商标、专利、版权、域名代理业务管理系统，能管理的来/去世界各个国家和地区的几乎所有商标、专利、版权、域名相关的业务。

虽然IPSys商标管理系统功能全面，但由于对整个系统进行了很好的架构，因此系统并不大，程序包只有6M左右,包括初始数据库包总共才10来M。

为了方便IPSys商标管理系统用户的安装和升级, 特别将IPSys商标管理系统制作成纯绿色软件。这样，安装时,客户端只须拷贝一个快捷方式即可, 完全不需要IT人员参与。升级时,只须覆盖共享文件夹中的程序文件，客户端用户根本无须参与。因此IPSys商标管理系统安装和升级都是十分方便的。

IPSys商标管理系统占用系统资源少，普通机器都可运行，为了更流畅运行，推荐的配置如下：

服务器端：

操作系统：Windows Server 2003 以上，推荐Windows Server 2008(RAID，卷影)

内存: 2G 以上

硬盘: 100G以上, 如果电子卷很大的话，需要更多的硬盘空间

数据库: SqlServer 2008以上(配置: a. 启用Sqlserver登陆，b. 新建sqlserver登录用户, c. 防火墙运行Sqlserver远程连接，允许1344端口即可)

客户端：

操作系统：Windows XP 以上

内存: 512M以上

硬盘: 10G以上

办公软件: Office 2003以上

杀毒软件: 推荐微软的MSE

 

IPSys商标管理系统安装过程如下: 

1. 拷贝IPSys商标管理系统安装包IPSys_Setup.rar到要安装的目录下, 如果是多人共同使用，应在局域网内设一个共享目录，将IPSys商标管理系统安装包IPSys_Setup.rar拷贝到该共享目录下。

2. 解压IPSys商标管理系统安装包IPSys_Setup.rar, 得到IPSys目录.

3. 将其中的DB目录剪切到数据库服务器上，并附加到SqlServer数据库中。

4. 右键IPSys目录内的IPSys.exe主程序, 选择创建快捷方式, 创建IPSys主程序的快捷方式, 我们可以改名为IPSys商标管理系统。

5. 将IPSys商标管理系统的快捷方式拷贝到用户桌面上。

6. 双击IPSys商标管理系统的快捷方式启动IPSys。第一次启动时由于未配置数据库，因此会弹出如下对话框，点击确定按钮，进入主界面。

7. 选择主菜单上的 工具->数据库配置，进入数据库配置界面。在其中输入在第3步中的数据库服务器所在的IP地址及用户名/密码，点确定, IPSys商标管理系统初始安装后的配置工作就完成了, 以后就不需要再配置了。

8. 直接点主菜单中的 文件->登录，或退出 IPSys商标管理系统，重新进入登录界面，输入初始管理员用户名/密码 Admin/Admin,点确定。

9. 进入主界面。

IPSys商标管理系统安装成功！

如果在安装IPSys商标管理系统的过程中出现任何问题，欢迎 到IPSys商标管理系统的论坛中发表。
