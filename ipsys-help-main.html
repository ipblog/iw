主界面布局

Rate this Content 0
     
0 投票

 

IPSys商标管理系统的主界面布局与Outlook类似，如下：

主界面各区的名称如下：

1. 应用程序菜单栏

   这是与整个IPSys商标管理系统相关的一些功能或设置，如登录、退出、界面的主题、界面的字体、数据库配置等。这里的操作与具体的实体数据无关，与具体的实体数据相关的操作均放在接下来的7.表格工具栏中。

2. 应用程序工具栏

   这里显示的是应用程序菜单栏中最常用的功能, 目前有常用查询、翻译、登录、退出。

3. 导航树

   这是整个程序实体数据操作的起点，所有的实体数据操作基本上都是从选中某一个树节点开始的。根据权限的不同，用户在这里看到的的节点是不同的，如管理员能看到所有节点，代理人仅能看到我的案件、我的业务等节点，财务人员登录仅能看到汇率、帐单节点。

4. 查询输入区

   这里显示的是导航树当前选中节点下数据的可用查询项，通过输入查询条件然后点击左下角的检索按钮对选中节点内的数据进行过滤查询。

5. 导航栏

   与Outook一样，这里可以切换到不同的导航树中。

6. 表格标签栏

   与多标签浏览器一样，通过表格标签栏可以打开多个表格,每个表格显示不同的检索结果。

7. 表格工具栏

   通过点击表格工具栏上不同的按钮对表格中选中的数据进行相应的操作。

8. 表格

   显示导航树当前选中节点下的数据。如果启用了分页功能，在表格下方还会有分页导航栏。

9. 状态栏

   在状态栏右侧会显示当前登录用户。

IPSys商标管理系统的功能归类到如下几大类:

  1. 工作区
  2. 案件管理
  3. 客户管理
  4. 人力资源
  5. 财务管理
  6. 系统配置
  7. 安全设置

每大类数据都放到同一个导航栏内，该类内的数据都以该导航栏内的导航树作为入口，数据的操作都显示在表格工具栏上。

IPSys商标管理系统采用的布局追求的是以尽可能少的操作步骤达到需要的目的，减少切换次数和点击次数，而且各种数据的操作保持高度一致。
